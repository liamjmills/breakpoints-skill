This page previews any URL, but it embeds your site cross-origin — so scroll-sync, synced Back/Forward, and Save PNG can’t work here (browsers block cross-frame access). Install the skill to run Breakpoints inside your own Claude Code: it proxies your local dev server same-origin, so every feature works.
